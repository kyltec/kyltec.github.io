Boomtown

Boomtown is a web application designed for posting and exchanging items. Both the server and user side had to be built for the app to work which was both frustrating and fun at the same time. Some features include a profile creation, live feedback of posted items, and posting a new item. A server is required to run the site so if you want you can create one and download the files from my github by clicking on the picture.
